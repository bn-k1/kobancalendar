# KobanCalendar

## 概要
本ツールは、交番表に基づいた勤務スケジュールを自動生成・管理するためのシステムです。  

特定の基準日を設定することで、X日周期のシフトローテーションを算出し、直感的なカレンダー表示で可視化できます。


## 交番表とは 
交番表とは、特定の作業を行う担当者の勤務順を決めたローテーション表 である。
各担当者には基準日を起点とした番号が割り振られ、毎日次の番号のシフトを担当する。
例えば、基準日に「83番」の勤務を担当した人は、翌日「84番」の勤務を担当する。
番号が最後まで進むと、翌日は最初の番号に戻る（循環する）。
各勤務には作業内容・開始時間・終了時間が記載され、個々のスケジュールが明確になる。
交番表を使うことで、組織全体の業務を円滑に回すことができる。


## 機能

- **基準日を設定し、CSVの行数周期でシフトを計算**
- **インタラクティブなカレンダー表示**
- **土曜・日曜・祝日・特定の勤務パターンを考慮**
- **日ごとの勤務情報をカラー分類**
  - **休日** → 赤
  - **特定勤務** → 黄
  - **通常勤務** → 青


## セットアップ

### 前提条件

- [Node.js](https://nodejs.org/)（バージョン14以上）
- [npm](https://www.npmjs.com/)または[yarn](https://yarnpkg.com/)

### インストール手順

1. リポジトリをクローンします。

   ```bash
   git clone https://github.com/bn-k1/kobancalendar.git
   cd kobancalendar
   ```

2. 依存関係をインストールします。

   ```bash
   npm install
   ```

3. `config.json`ファイルを設定します。

   ```json
   {
     "BASE_DATE": "YYYY-MM-DD",
     "custom_holidays": ["08/15", "12/30"]
   }
   ```

   - `BASE_DATE`: シフト計算の基準日（YYYY-MM-DD形式）
   - `custom_holidays`: 独自に設定する休日の配列

4. `data/`のCSVファイル（`weekday.csv`:平日,`saturday.csv`:土曜,`holiday.csv`:日祝）を編集します。フォーマットは以下のとおりです。

   ```csv
   早番,08:00,16:00
   遅番,16:00,00:00
   夜勤,00:00,08:00
   ```

5. アプリケーションを起動します。

   ```bash
   npm start
   ```

   ブラウザで`http://localhost:3000`にアクセスすると、カレンダーが表示されます。


## 注意点

本リポジトリでは、例として別の銀河の小惑星ゾルガンタスの宇宙警備隊のパトロール用交番表の情報を用いています。

地球上の特定の組織や企業とは関係ありません。

CSVの行数は交番表の最後のコマの数字と一致する必要があります。

CSVファイルの行数がすべて一致していないと正しく動作しません。
